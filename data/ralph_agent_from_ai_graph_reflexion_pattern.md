# Reflexion Pattern Integration for `chat.py`

## Overview  

The Reflexion pattern (Shinn 2023) is a lightweight self‑critique loop that lets an LLM improve its own reasoning over time. In the Quran Knowledge Graph project we embed this pattern directly into the conversational agent (`chat.py`). The goal is to reduce the “weak‑question” cluster (e.g., repeated meditation/reverence queries about Surah 55) by having the agent remember what it would change after each turn and then surface those memories when a similar query re‑appears.

## High‑Level Flow  

1. **Agent Turn Completion** – After the agent produces a final answer (including any tool calls), we invoke a *Reflexion step*.  
2. **Generate Memo** – The LLM is prompted with the full turn transcript and asked:  
   > “In one sentence, what would I do differently if I had to answer this again?”  
   The response is stored as a **Reflexion memo**.  
3. **Persist Memo** – Memos are appended to a persistent JSONL store (`reflexion_memos.jsonl`). Each entry contains:  
   * `query_hash` – SHA‑256 of the user query (canonicalized).  
   * `memo` – the one‑sentence reflexion.  
   * `timestamp` – ISO‑8601 time of creation.  
   * `embedding` – vector of the original query (via the existing embedding model).  
4. **Recall on Similar Query** – When the agent receives a new user query, the tool `tool_recall_similar_query` is invoked early in the planning stage. It:  
   * Computes the embedding of the incoming query.  
   * Performs a cosine‑similarity search against stored embeddings (k‑nearest, default k = 5).  
   * Retrieves the most relevant memos (by similarity score).  
   * Returns a concatenated string of up to three top memos.  
5. **Inject into Planner Context** – The planner prompt is extended with a *“Reflexion Context”* section that contains the retrieved memos. This gives the planner immediate awareness of past self‑critiques, steering it away from previously identified pitfalls (e.g., over‑reliance on `tool_search_quran` for meditation questions).

### 1. `chat.py` – Add Reflexion Hook  

```python
import hashlib, json, time
from pathlib import Path
from utils.embeddings import embed_text   # existing embedding helper
from utils.vector_store import cosine_search  # thin wrapper around numpy

REFLEXION_STORE = Path("data/reflexion_memos.jsonl")

def _hash_query(text: str) -> str:
    return hashlib.sha256(text.strip().lower().encode()).hexdigest()

def _store_reflexion(query: str, memo: str) -> None:
    entry = {
        "query_hash": _hash_query(query),
        "memo": memo,
        "timestamp": time.strftime("%Y-%m-%dT%H:%M:%SZ", time.gmtime()),
        "embedding": embed_text(query).tolist(),
    }
    REFLEXION_STORE.parent.mkdir(parents=True, exist_ok=True)
    with REFLEXION_STORE.open("a", encoding="utf-8") as f:
        f.write(json.dumps(entry, ensure_ascii=False) + "\n")

def _recall_similar_memos(query: str, k: int = 5) -> list[str]:
    if not REFLEXION_STORE.exists():
        return []
    query_emb = embed_text(query)
    candidates = []
    with REFLEXION_STORE.open("r", encoding="utf-8") as f:
        for line in f:
            entry = json.loads(line)
            candidates.append(entry)
    # compute similarities
    sims = [
        (cosine_search(query_emb, entry["embedding"]), entry["memo"])
        for entry in candidates
    ]
    sims.sort(key=lambda x: x[0], reverse=True)
    # return top‑3 unique memos
    top_memos = []
    for _, memo in sims[:k]:
        if memo not in top_memos:
            top_memos.append(memo)
        if len(top_memos) >= 3:
            break
    return top_memos
```

### 2. Planner Prompt Augmentation  

```python
def _build_planner_prompt(user_query: str, tool_context: str) -> str:
    reflexion_context = ""
    similar_memos = _recall_similar_memos(user_query)
    if similar_memos:
        reflexion_context = (
            "Reflexion Context (past self‑critiques relevant to this query):\n"
            + "\n".join(f"- {m}" for m in similar_memos)
            + "\n\n"
        )
    base_prompt = f"""You are an expert Quran knowledge‑graph agent.
{reflexion_context}
User query: {user_query}
Relevant tool output: {tool_context}
...
"""
    return base_prompt
```
