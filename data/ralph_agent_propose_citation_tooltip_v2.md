## 3. JavaScript Hook (high‑level)

```js
// Called on verse hover (existing handler)
function showTooltip(verseId, anchorEl) {
  const tooltip = document.querySelector('.verse-tooltip');
  tooltip.dataset.verseId = verseId;

  // Pull data from the already‑cached verse object
  const verse = window.chatDonePayload.verses[verseId]; // {arabicText, english}
  tooltip.querySelector('.arabic-text').textContent  = verse.arabicText;
  tooltip.querySelector('.english-text').textContent = verse.english;

  // Position near anchorEl
  const rect = anchorEl.getBoundingClientRect();
  tooltip.style.top  = `${rect.bottom + window.scrollY + 6}px`;
  tooltip.style.left = `${rect.left + window.scrollX}px`;
  tooltip.style.display = 'block';
}

// Pin / un‑pin logic
document.querySelector('.pin-btn').addEventListener('click', e => {
  const tt = e.currentTarget.closest('.verse-tooltip');
  tt.classList.toggle('pinned');
  if (tt.classList.contains('pinned')) {
    tt.style.position = 'fixed';
    tt.style.top = '20px';
    tt.style.right = '20px';
    tt.style.left = 'auto';
  } else {
    tt.style.position = 'absolute';
  }
});
```

* The existing **/chat** “done” event already ships each verse’s `arabicText` and `english`. No extra API call is required.
