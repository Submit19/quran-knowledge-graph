# Quran Contains All The Examples We Need

- 70-
QURAN CONTAINS A LL THE EXAMPLES WE NEED
There are four scattered verses stating generally that
Quran contains all the examples, allegories, or similit
udes that we need. These verses are 17:89, 18:54, 30:
58, & 39:27. By adding the verse numbers of these four
crucial verses, the total is 228, or 19 x 12.
