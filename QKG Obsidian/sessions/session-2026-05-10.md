## Key findings worth remembering

- **The 647s outlier is Python-side, not DB-side.** Don't waste more time profiling Cypher.
- **Aura Agent locks to Gemini + OpenAI/Google embeddings.** Not a viable migration target while we're on BGE-M3.
- **Our 21-tool MCP overhead is ~4,200 tokens before first message.** 62% reduction possible with 8-tool startup set + 3 discoverable categories, ranked by `RETRIEVED` edge counts (zero new infra). See [[research/from_neo4j_yt_mcp_balanced_tool_grouping]].
- **Abstract-concept query weakness is a routing failure, not retrieval.** Two videos independently confirm. Fix: complexity-router → `concept_search` first. See [[research/from_neo4j_yt_router_agent]].
- **Reasoning trace schema gaps: `:ReasoningStep` between Trace and ToolCall, bitemporal `RETRIEVED` edges, nightly consolidation "sleep step".** Triple-confirmed across research streams.
