# SESSION_LOG.md

Append-only log of Claude Code sessions on the QKG project.

**Format:** one block per session, newest at top. Each block:
- Heading with date + Claude variant + a 1-line goal
- "Shipped" — concrete artefacts/commits
- "Queued" — what's pending for next time
- "Open questions / decisions made" — anything worth remembering
- "Tip for the next session" — one line

active window) appends here. New sessions read the top entry first.

Pair this with `STATE_SNAPSHOT.md` (auto-generated by `scripts/state_snapshot.py`)
and the most recent note in `QKG Obsidian/sessions/`.
